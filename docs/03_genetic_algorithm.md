# Genetic Algorithm

## Evolution Loop

1. Evaluate population
2. Compute fitness
3. Select parents
4. Crossover
5. Mutation
6. Next generation

---

## Fitness Modes

### Individual Fitness

Each agent evaluated independently.

### Tribe-Level Fitness

Fitness shared across all agents in the tribe.

This enables group-level strategies.

---

## Key Parameters

- population size
- mutation rate
- crossover rate
- number of generations
- evaluation episodes
